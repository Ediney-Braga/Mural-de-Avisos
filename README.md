# Mural de Avisos!
![Captura de tela de 2022-09-25 23-00-08](https://user-images.githubusercontent.com/103907308/192179798-9903f82d-8695-424c-9e28-8c96fa4b9d64.png)


  Este é um mural de avisos que tem como objetivo o estudo da integração do Front-end com o Back-end. 
Para isso utilizei o Node.js e o Express, que auxiliou na criação das rotas, na utilização dos metodos 
GET e POST.

### Tecnologias utilizadas:

- HTML5
- CSS3
- JavaScript
- Node.Js
- Express
